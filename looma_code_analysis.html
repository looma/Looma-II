Looma Codebase Analysis What is Looma? Looma is an offline educational platform designed for schools in Nepal. It runs on low-cost ARM-based computers (Odroid) serving a local network of tablets. It provides textbook-aligned content (videos, PDFs, lessons, games, dictionaries) for grades 1–12 in Nepali and English, functioning without an internet connection.
